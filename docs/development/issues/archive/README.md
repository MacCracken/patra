# Archived Issues

Resolved upstream-tracking issues — cyrius bugs and cross-target ABI gaps — that surfaced during patra development. Each archived file keeps its original body verbatim plus an `ARCHIVED YYYY-MM-DD — RESOLVED …` block at the top. Don't rewrite history here — the archive's value is being a faithful record of what was filed and what fixed it.

Open issues sit at the parent [`docs/development/issues/`](../) directory. Issues move here when the upstream fix lands and the workaround is no longer load-bearing.
